# Exploratory Data Analysis of Passenger Survival Patterns (Titanic Dataset)

## Project Overview
This project explores historical passenger data from the Titanic to identify key factors influencing survival outcomes. The focus is on demonstrating **data science skills**, including data cleaning, feature engineering, visualization, and predictive modeling not the Titanic itself.  

In real-life applications, this kind of analysis translates to:
- Risk modeling
- Customer segmentation
- Predictive analytics
- Feature importance analysis

## Tools & Technologies
- **Python**  
- **Pandas & NumPy** – Data manipulation  
- **Matplotlib & Seaborn** – Data visualization  
- **Scikit-learn** – Logistic Regression & model evaluation  

## Key Insights & Results
- Overall survival rate: 38.38%  
- Females had a 74% survival rate vs. 19% for males  
- First-class passengers survived the most  
- Logistic Regression model accuracy: 81.56%  
- Key features influencing survival: sex, passenger class, fare, and age
